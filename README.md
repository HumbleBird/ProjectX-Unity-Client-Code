# ProjectX - Unity Client Code

본 저장소는 Unity 기반 모바일 액션 RPG 프로젝트의
클라이언트 핵심 로직(Scripts)만 발췌한 코드 포트폴리오입니다.

## Environment
- Unity 2022.3 LTS
- C#
- Target: Mobile

## Highlights
- GameEntity 중심 아키텍처 설계
- 전투 / 장비 / 투사체 로직 분리
- Grid 기반 이동 및 경로 탐색 최적화
- 라이브 서비스 고려한 구조 개선

## Folder Guide
- Scripts/Core       : 공통 시스템
- Scripts/Combat    : 전투 로직
- Scripts/Grid      : 이동 및 경로 탐색
- Scripts/GameEntity: 유닛 기반 구조

※ 실행 목적이 아닌 코드 리뷰용 저장소입니다.
